# Power-Up Sequence Diagram Designer

This is an online power-up/power-down sequence diagram constructor for hardware engineers. It started basic, but has evolved into a highly capable and robust timeline tool to get the job done accurately.
There are no subscriptions, no ads, and everything runs locally in your browser.

### ✨ Key Features
* **Save & Load Configurations:** Export and import your sequence setups as clean, human-readable `.json` files.
* **Custom Trigger Logic:** Accurately track parent signal slopes with a custom 0–100% trigger input.
* **Advanced Delays:** Separate your base propagation delays from intentional added delays ("Add. Dly").
* **Dynamic Timeline Grid:** A mathematical background grid auto-scales to your sequence and intelligently jumps across visual "Time Breaks".
* **Total Elapsed Time:** Automatically calculates the true logical duration of your sequence from the first edge to the last edge.
* **Datasheet Ready:** Export directly to vector SVG, featuring a built-in Black & White (B&W) mode for formal documentation.
* **Auto-Cloning:** Instantly reverse your power-up logic into a power-down sequence with a single click.
